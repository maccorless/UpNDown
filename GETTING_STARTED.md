# Getting Started

This file is a quick-start companion to `README.md`.

## Install

```bash
npm install
```

## Run Locally

Terminal 1:

```bash
npm run dev --workspace @upndown/server
```

Terminal 2:

```bash
npm run dev --workspace @upndown/client -- --host 127.0.0.1 --port 5173
```

Open:
- `http://127.0.0.1:5173`

## Test

```bash
npm run test
```

## Build

```bash
npm run build
```
